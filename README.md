# 🔥 Tri-Gomoku 三人五子棋

一个支持三人轮流对战的五子棋网页游戏。

## 🎮 在线试玩

👉 [点击开始游戏](https://你的用户名.github.io/tri-gomoku)

## ✨ 特性

- **三人轮流对战** - 红/蓝/绿三色棋子
- **大棋盘** - 支持 19×19 / 25×25 / 29×29 三种规格
- **缩放与拖拽** - 支持鼠标滚轮缩放、右键拖拽
- **移动端适配** - 双指缩放、单指拖拽、点击落子
- **音效系统** - 落子、胜利、悔棋都有音效反馈
- **胜负判定** - 支持 "≥5子获胜" 或 "恰好5子" 规则

## 🎯 游戏规则

1. 三名玩家轮流落子（红 → 蓝 → 绿 → 红...）
2. 率先形成连续 5 子或更多者获胜
3. 棋盘下满无胜者则为平局

## 📱 操作说明

| 平台 | 操作 |
|------|------|
| PC | 左键落子 / 滚轮缩放 / 右键拖拽 |
| 移动端 | 点击落子 / 双指缩放 / 单指拖拽 |

## 🚀 本地运行

直接用浏览器打开 `index.html` 即可。
